🔧 แผนงาน / แบ่งทีม

แบ่งทีมช่าง 2–4 คนทำงานขนาน — ลดเวลาติดตั้งให้เร็วที่สุด

ทีม A — เดินสายเวที + จุดรับสัญญาณ 4 จุด (2 คน)

หน้าที่: ต่อสายจากไมค์/เครื่องดนตรี → จุดรับสัญญาณ 4 จุด + Drum Multicore

  1. วางมัลติคอร์สั้น (จุดที่ 1) ที่ขอบเวทีซ้ายใกล้มิกเซอร์ (~3m)
  2. วาง Stage Box ฝั่งซ้าย (จุดที่ 2) ตำแหน่ง เบส/กีตาร์โปร่ง/นักร้องนำ
  3. วาง Stage Box ฝั่งขวา (จุดที่ 3) ตำแหน่ง กีตาร์ไฟฟ้า/นักร้อง #2
  4. วาง Drum Multicore ใกล้กลองชุด
  5. ต่อ DI Box 4 ตัว: คีย์บอร์ด L+R (2) → จุดที่ 1 / เบส (1) + กีตาร์โปร่ง (1) → จุดที่ 2
  6. เดินสาย XLR จากไมค์/DI เข้าจุดรับสัญญาณ ตาม กลุ่ม A–C
  7. เดินสาย XLR 7 เส้นจากไมค์กลอง → Drum Multicore ตาม กลุ่ม D
  8. ลาก Snake จากจุดที่ 2 (~20m) + จุดที่ 3 (~15m) → ตำแหน่ง X Air 18
  9. ลาก Snake จาก Drum Multicore → ตำแหน่ง ui12 (~15m)
สายที่ต้องเตรียม: XLR สั้น ~19 เส้น (A1, B1, C1, D1) + Snake 4 เส้น (A2, B2, C2, D2) + TRS/TS 2 เส้น (KB→DI) + DI Box 4 ตัว

ทีม B — ต่อ ui12 + X Air 18 + เชื่อมระบบ (1–2 คน)

หน้าที่: ตั้ง Mixer ทั้ง 2 ตัว เชื่อมเข้าด้วยกัน

  1. วาง SoundCraft ui12 ใกล้ Drum Multicore
  2. เสียบ Snake จาก Multicore เข้า ui12 Ch1–7
  3. เสียบ 4 สาย จาก ui12 → X Air 18 ตาม กลุ่ม E:
    • E1: Aux 1 (Kick) → X Air Ch1 — สาย TRS→XLR
    • E2: Aux 2 (Snare) → X Air Ch2 — สาย TRS→XLR
    • E3: Main L (Tom+OH) → X Air Ch3 — ⚠️ สาย XLR Male-to-Male
    • E4: Main R (Tom+OH) → X Air Ch4 — ⚠️ สาย XLR Male-to-Male
  4. เสียบ Snake จากจุดที่ 2 (ฝั่งซ้าย) เข้า X Air 18 Ch5, 6, 10, 13, 14
  5. เสียบ Snake จากจุดที่ 3 (ฝั่งขวา) เข้า X Air 18 Ch7, 11, 12
  6. เสียบมัลติคอร์สั้น (จุดที่ 1) เข้า X Air 18 Ch8, 9, 15, 16
  7. เปิด +48V สำหรับ ไมค์ Overhead (Ch6-7 บน ui12) — ห้ามเปิด +48V สำหรับ Dynamic Mic
  8. ตั้ง WiFi ของ X Air 18 → ต่อ Tablet/มือถือเข้าแอป X Air Edit
สายที่ต้องเตรียม: TRS→XLR 2 เส้น (E1-E2) + XLR Male-to-Male 2 เส้น (E3-E4)

ทีม C — แขวนลำโพงในอาคาร + Marani (1–2 คน)

หน้าที่: ติดตั้งลำโพง FOH ในอาคาร + เชื่อม Marani

  1. แขวน/ตั้ง Yamaha DXR ×4 ใบ: ซ้าย 2 ใบ (ใบแรก+Link) + ขวา 2 ใบ (ใบแรก+Link)
  2. ตั้ง Sub MA118 ×2 ใบ: ซ้าย 1 + ขวา 1 (วางพื้นใกล้เวที)
  3. วาง Marani DPA260p+ ใกล้ X Air 18
  4. สาย F1–F2: X Air Main L+R → Marani Input (XLR ×2 เส้น ~3m)
  5. สาย F3: Marani Out 1 → DXR ซ้ายใบแรก (XLR ~10m)
  6. สาย F3-Link: DXR ซ้ายใบแรก Link Out → DXR ซ้ายใบสอง (XLR ~1–2m)
  7. สาย F4: Marani Out 2 → DXR ขวาใบแรก (XLR ~10m)
  8. สาย F4-Link: DXR ขวาใบแรก Link Out → DXR ขวาใบสอง (XLR ~1–2m)
  9. สาย F5: Marani Out 3 → Sub MA118 ซ้าย (XLR ~8m)
  10. สาย F6: Marani Out 4 → Sub MA118 ขวา (XLR ~8m)
สายที่ต้องเตรียม: XLR 8 เส้น (F1–F6 + F3-Link + F4-Link) — ไม่มี Speakon (ลำโพง Active ทั้งหมด ต่อ XLR)

ทีม D — ลำโพงนอกอาคาร + dbx 260 (1–2 คน)

หน้าที่: ติดตั้งลำโพงนอกอาคาร + เชื่อม dbx DriveRack 260

  1. ตั้ง Wharfedale Pro Titan 12 ×2 ใบ ที่ตำแหน่งนอกอาคาร โซน Titan
  2. ตั้ง VL-AUDIO MT-65B ×3 ใบ (Passive) พ่วง Speakon Daisy-chain: Power Amp → ใบ 1 → Link ใบ 2 → Link ใบ 3
  3. วาง dbx DriveRack 260 + Power Amp ในแร็คเดียวกัน ใกล้ X Air 18
  4. สาย G1: X Air Aux 6 → dbx 260 Input 1 (TRS→XLR ~5m)
  5. สาย G2: dbx Out 1 → Titan 12 ×2 ใบ (XLR ~10m — แยกเส้น)
  6. สาย G3: dbx Out 2 → Power Amp Input (XLR ~1m)
  7. สาย G4: Power Amp Output → VL-AUDIO ใบแรก (Speakon ~15m)
  8. สาย G4-Link1: VL-AUDIO ใบ 1 Link → ใบ 2 (Speakon ~2m)
  9. สาย G4-Link2: VL-AUDIO ใบ 2 Link → ใบ 3 (Speakon ~2m)
สายที่ต้องเตรียม: TRS→XLR 1 เส้น (G1) + XLR 3 เส้น (G2, G3) + Speakon 3 เส้น (G4, G4-Link1, G4-Link2)

ทีม E — In-Ear Monitor (ทำหลังทีม B เสร็จ)

หน้าที่: เดินสาย In-Ear Monitor 5 จุดบนเวที ผ่าน Snake Return

  1. H1: X Air Aux 1 → จุดที่ 1 Snake Return → In-Ear คีย์บอร์ด (TRS→XLR ~3m)
  2. H2: X Air Aux 2 → จุดที่ 2 Snake Return → In-Ear เบส (TRS→XLR ~20m)
  3. H3: X Air Aux 3 → ⚡ Y-split → จุดที่ 2 (In-Ear Vocal 1) + จุดที่ 3 (In-Ear Vocal 2) — นักร้อง 2 คนแชร์ Mix เดียวกัน
  4. H4: X Air Aux 4 → จุดที่ 3 Snake Return → In-Ear กีตาร์ (TRS→XLR ~15m)
  5. H5: X Air Aux 5 → Drum Multicore Return → In-Ear กลอง (TRS→XLR ~15m)
สายที่ต้องเตรียม: TRS→XLR 5 เส้น (H1–H5) + Y-split XLR 1 ตัว (Aux 3 แยก 2 ทาง)

🎚 ปรับจูน & เปิด-ปิดระบบ

📱 ลำดับเปิดระบบ (Power On)

  1. เปิด ui12 ก่อน (Mixer กลอง) — รอ WiFi พร้อม
  2. เปิด X Air 18 (Mixer หลัก) — รอ WiFi พร้อม
  3. เปิด Marani DPA260p+ (DSP Crossover ในอาคาร)
  4. เปิด dbx DriveRack 260 (โปรเซสเซอร์นอก)
  5. เปิด Power Amp นอกอาคาร (ขับ VL-AUDIO Passive)
  6. เปิด DXR + Sub MA118 (ลำโพง Active ในอาคาร — เปิดทีหลังสุด)
  7. เปิด In-Ear Monitor

หลักการ: เปิดจาก Source → Mixer → Processor → แอมป์/ลำโพง (เรียงจากต้นทางไปปลายทาง)

🔴 ลำดับปิดระบบ (Power Off) — กลับกัน

  1. ปิด In-Ear Monitor
  2. ปิดลำโพง DXR + Sub MA118 (ปิดก่อน)
  3. ปิดแอมป์ลำโพงนอก / ปิด dbx 260
  4. ปิด Marani
  5. ปิด X Air 18
  6. ปิด ui12

หลักการ: ปิดจากปลายทาง → กลับมาต้นทาง (ลำโพงปิดก่อน ป้องกันเสียง Pop)

🏛 ปรับจูน Marani DPA260p+ (ในอาคาร — DSP Crossover)

  1. Crossover: ตั้ง ~80 Hz — ต่ำกว่า 80 Hz ส่ง Sub MA118 / สูงกว่า 80 Hz ส่ง DXR
  2. EQ: ปรับตามห้อง — ถ้าเสียงก้อง ลด Mid 400-800Hz / ถ้าทุ้มเกิน ลด 100-200Hz
  3. Limiter: เปิดทุก Output — ป้องกันลำโพงพัง ตั้ง Threshold ตาม Spec ลำโพง
  4. Output Level: ปรับให้ซ้าย-ขวาเท่ากัน (ใช้ SPL Meter หรือฟังจากตรงกลางห้อง)

🎛 ปรับจูน dbx DriveRack 260 (นอกอาคาร)

  1. Input Gain: ตั้งให้สอดคล้องกับ Level ที่ส่งมาจาก Aux 6 ของ X Air 18 (เช็ค Level Meter)
  2. Out 1 (Titan 12): Full range — EQ ปรับตามสภาพแวดล้อม ตั้ง Limiter
  3. Out 2 (VL-AUDIO MT-65B): Full range หรือ High-pass ถ้าขนาดเล็ก ตั้ง Limiter
  4. Delay: ตั้ง Delay ถ้าลำโพงนอกมองเห็นเวทีได้ (ให้เสียงตรงกับในร้าน)

🎛 ปรับจูน X Air 18 (มิกเซอร์หลัก)

  1. Gain Structure: ตั้ง Gain แต่ละ Ch ให้ Level เฉลี่ยอยู่ที่ -18 ถึง -12 dBFS (ขีดเขียว)
  2. Phantom +48V: เปิดเฉพาะ Ch ที่ใช้ Condenser Mic — ห้ามเปิดกับ Dynamic Mic
  3. Main Mix: ปรับ Level ทุก Ch ให้ได้มิกซ์ที่ต้องการ Main L+R → ส่ง FOH ในร้าน
  4. Aux 6 (นอกอาคาร): ปรับ Send Level แต่ละ Ch ตามความต้องการ — สามารถลดเหลือ 0 เพื่อปิดเสียงนอก
  5. Aux 1–5 (In-Ear Monitor): ปรับ Mix ให้แต่ละนักดนตรีได้ยินเสียงที่ต้องการ
    • Aux 1 (KB): เน้นคีย์บอร์ด + Vocal
    • Aux 2 (เบส): เน้นเบส + Kick + Vocal
    • Aux 3 (ร้อง ×2 Y-split): เน้นร้อง + เครื่องดนตรีทุกอย่างเบาๆ — นักร้อง 2 คนแชร์ Mix เดียวกัน
    • Aux 4 (กีตาร์): เน้นกีตาร์ + Vocal
    • Aux 5 (กลอง): เน้น Click Track (Ch16) + Vocal + เบส (กลองได้ยินของตัวเองอยู่แล้ว)

🥁 ปรับจูน ui12 (กลอง Pre-mix)

  1. Aux 1 Send: ส่งเฉพาะ Kick (Ch1) → X Air Ch1 — ห้ามส่ง Ch อื่น
  2. Aux 2 Send: ส่งเฉพาะ Snare (Ch2) → X Air Ch2 — ห้ามส่ง Ch อื่น
  3. Main L+R: ส่ง Tom 1-3 + OH L+R → X Air Ch3-4 (Stereo)
  4. Phantom +48V: เปิดเฉพาะ Ch6 (OH L) และ Ch7 (OH R) — ไมค์ Condenser
  5. EQ: Kick ตัด ~300Hz เพิ่ม ~60Hz / Snare เพิ่ม ~200Hz + ~5kHz / Tom ตัด ~400Hz
  6. Gate: ถ้ามี เปิด Gate บน Kick/Snare/Tom — ลด Bleed จากเสียงอื่น

📋 เช็คลิสต์ก่อนเริ่มงาน

  • ✅ ทุกสายต่อแน่น ไม่หลวม — ดึงทดสอบทีละเส้น
  • ✅ Channel Assignments ตรงกับตารางเดินสาย — ลองพูดไมค์ทีละตัว
  • ✅ Phantom +48V เปิดเฉพาะ Condenser (OH L/R) — ห้ามเปิดกับ Dynamic
  • ✅ ไม่มี Feedback — ถ้ามี ลด Gain แล้วเช็ค In-Ear ที่ไม่สวม
  • ✅ ปิดเสียงนอกอาคาร (Aux 6 = 0) ก่อน Soundcheck — เปิดทีหลังเมื่อพร้อม
  • ✅ ลำโพง DXR Link Out ต่อถูกทิศ — ใบแรกต่อ Marani ใบที่สองต่อ Link
  • ✅ Metronome (Ch16) ห้ามส่งออก FOH — ส่งออก In-Ear (Aux 5) เท่านั้น
